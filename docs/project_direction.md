# Project Direction

## Core Goal

Build a long-term AI alignment and evaluation laboratory.

## Focus Areas

1. Safe refusal behavior
2. Over-refusal detection
3. Rule following
4. Boundary awareness
5. Hallucination detection
6. Uncertainty calibration
7. Tool-use safety
8. Defensive cybersecurity evaluation

## Long-Term Vision

1. Build evaluation infrastructure.
2. Build or fine-tune a small model.
3. Use the evaluation framework to improve alignment.
4. Test safe tool use in controlled environments.
5. Develop a defensive cybersecurity assistant for sandboxed training environments.

## Guiding Principle

The project focuses on reliability, alignment, and safe deployment rather than offensive capability.
